Bandcamp Use Case 1.13: Search For Music

Actors: Member

Scope: A social music sharing system

Level: User goal

Precondition: Member is logged into system

Post-condition: Returns matching results

Main Success Scenario:
Steps:

  1. Member types song title into search bar.
  2. Member submits search.
  3. System looks for matching song titles.
  4. All found matches are displayed to member.

Alternative Scenarios:
Steps:

  • 4a1. Member types song title into search bar.
  • 4a2. Member submits search.

Priority: High

Frequency of Use: High

Open Issues:
Will the system display results if the member enters 1 character only?
Will the system check for special characters?
Will the system search for music event?
Will the system search for users?
Could unregistered user be allowed to search?
