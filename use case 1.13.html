1.13 Search For Music

Actors: Registered User, Music System

Scope: A social music sharing system

Level: User goal

Description: After a registered user logs in, they type in the name of a song they wish to look for into the search bar. As the user types the system presents a few results as hints below the search bar. Each hint will have an icon that will show the user what the search result relates to for example songs, registered users or comments. The user may select a hint or continue typing. When the user has finished typing or selects a hint and hits [Enter], they will be directed to a page of results.

The results page will list each song that contains the search text in its title, genre, description, or in its comments. If the registered user is taken to this results page as a result of clicking on one of the hints, the song related to that hint will be shown as the first result.
